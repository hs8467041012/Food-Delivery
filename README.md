Food Delivery Website
This repository contains a simple food delivery website built using HTML and CSS.
It is a static frontend project that demonstrates the layout and design of a basic online food ordering platform.

 Project Overview
The Food Delivery website showcases a clean and responsive landing page for a food ordering service.
It includes sections such as home, services, and contact, making it a good project for learning frontend basics.

 Features:
 Simple and clean UI
 Responsive design for mobile devices
 Food images and banners
 Fast-loading static website

Technologies Used:
HTML5
CSS3
